纠正：
1...Integer最开始不是null吗？怎么还有其他情况

代码：

order.setRenewCount(
    order.getRenewCount() == null ? 1 : order.getRenewCount() + 1
);

意思是：

如果 renewCount 是 null
   → 设置为 1
否则
   → 原来 +1

为什么这样写？

原因是数据库可能出现两种情况：

情况1：旧数据

有些历史数据：

renewCount = null
情况2：正常数据

申请借阅时已经写了：

renewCount = 0

代码：

order.setRenewCount(0);

所以：

原值	续借后
null	1
0	1
1	2

不过因为系统限制：

>=1 就不允许

所以实际上最多：

1
